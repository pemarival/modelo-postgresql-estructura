# 10_rollback

Esta carpeta contiene scripts de rollback alineados al modelo de datos actual.

Ejecucion sugerida:

1. Ejecutar funciones/procedimientos/vistas rollback.
2. Ejecutar rollback de tablas.
3. Ejecutar rollback de schemas.
4. Ejecutar rollback de extensiones.

Puedes ejecutar todo el flujo con el changelog raiz:

- `01_ddl/10_rollback/0000changelog.yaml`
